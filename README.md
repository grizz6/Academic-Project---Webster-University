# Academic-Project---Webster-University

R-based analytics practicum projects completed at Webster University. All scripts are written in R and cover data cleaning, EDA, and machine learning.

Projects

Project 1 — Used Device Price Analysis
Data: used_device_data.csv
EDA and predictive modeling on a used smartphone dataset. Covers missing value imputation, outlier handling, correlation analysis, and regression/classification models to predict normalized device prices.

Project 2 — Software Mailing List (NorthPoint)
Data: Software_Mailing_List.csv
Customer analytics on a software mailing list. Analyzes purchase behavior, spending distributions, and source channel performance. Includes clustering and classification models to predict purchase likelihood.

Project 3 — Mortgage Payback Prediction
Data: Mortgage.csv
Binary classification on mortgage loan data to predict default risk. Uses Random Forest, XGBoost, GBM, LASSO, and logistic regression with ROSE oversampling to handle class imbalance.
Final Project — Combined Practicum

Consolidated script combining all three datasets and analyses into a single end-to-end practicum. Includes all preprocessing, EDA, and modeling from Projects 1–3, plus neural network models (neuralnet).
